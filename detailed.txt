**Lines 58-93: Attraction & Repulsion Physics**
```tsx
const rect = element.getBoundingClientRect();
const centerX = rect.left + rect.width / 2;
const centerY = rect.top + rect.height / 2;

const deltaX = centerX - clientX;
const deltaY = centerY - clientY;
const distance = Math.sqrt(deltaX * deltaX + deltaY * deltaY);
```
**Physics Calculation:**
1. Element ka center nikala
2. Cursor se element center tak ka distance (Pythagorean theorem)
3. Agar distance < attractionRadius (80% of element size):
   - Attraction apply karo (cursor ko center ki taraf khicho)
4. Agar distance < 30px:
   - Repulsion apply karo (overlap prevent karne ke liye)

**Lines 73-85: Attraction/Repulsion Force**
```tsx
if (distance < repulsionRadius) {
    // REPULSION
    const repulsionForce = (repulsionRadius - distance) / repulsionRadius;
    finalX = clientX - (deltaX / distance) * repulsionForce * 20;
} else {
    // ATTRACTION
    const attractionForce = 0.15;
    finalX = clientX + deltaX * attractionForce;
}
```
**Kyu:** 
- Repulsion = Jab bahut paas hai, to cursor ko thoda door push karo
- Attraction = Jab moderate distance par hai, to cursor ko element ki taraf khicho

## üìã Summary (Overview)
Aapke website mein ek **custom magnetic cursor** implement kiya gaya hai jo mouse ke movement ko smooth banata hai aur interactive elements (buttons, links) ke paas magnetic attraction aur repulsion effect deta hai.

## üÜï Nayi File Banai Gayi

### `src/components/CustomCursor.tsx`

**Kya hai ye?**
Ye ek custom cursor component hai jo default mouse cursor ko replace karta hai ek beautiful, glowing, magnetic cursor se.

**Kyu banaya?**
- Modern websites (jaise Apple, Awwwards winners) mein aisa cursor hota hai
- User experience ko premium feel dene ke liye
- Interactive elements ko highlight karne ke liye
- Website ko visually attractive banane ke liye

**Kaise banaya? (Technologies/Concepts Used)**

1. **React Hooks**
   - `useState` - cursor visibility, hovering state track karne ke liye
   - `useRef` - cursor DOM element ko reference karne ke liye
   - `useEffect` - mouse events ko listen karne ke liye

2. **Framer Motion** (Animation Library)
   - `useMotionValue` - mouse position ko instantly track karta hai
   - `useSpring` - smooth, physics-based movement ke liye (inertia effect)
   - `motion.div` - animated cursor circle aur trail banane ke liye
   - Spring Physics Parameters:
     * damping: 30 (kitna resistance hai movement mein)
     * stiffness: 200 (kitna tight/responsive hai)
     * mass: 0.5 (cursor ka "weight")

3. **Physics-Based Magnetic Effect**
   
   **Attraction Logic:**
   ```
   - Jab cursor kisi magnetic element ke 80% radius ke andar aata hai
   - Cursor ko element ke center ki taraf khichta hai
   - Force = 0.15 √ó distance (15% attraction strength)
   ```
   
   **Repulsion Logic:**
   ```
   - Jab cursor element ke bahut paas (30px se kam) aata hai
   - Cursor ko element se door dhakelta hai (overlap nahi hone deta)
   - Repulsion force dynamically calculate hota hai distance ke basis par
   ```

4. **DOM Query aur Data Attributes**
   - `document.querySelectorAll('[data-magnetic]')` - sabhi magnetic elements find karta hai
   - Real-time distance calculation har element ke liye
   - `getBoundingClientRect()` - element ki position/size calculate karne ke liye

5. **Responsive Design**
   - Touch devices par auto-hide (mobile par system cursor hi rahega)
   - `window.matchMedia('(pointer: coarse)')` - touch device detect karta hai

**Code Flow:**
```
Mouse Move ‚Üí 
Find Magnetic Elements ‚Üí 
Calculate Distance ‚Üí 
Apply Attraction/Repulsion Force ‚Üí 
Update Cursor Position with Spring Animation ‚Üí 
Render Cursor + Trail
```

## üéØ Overall Concept

### Magnetic Cursor Kya Hai?
Ek cursor jo:
1. **Smooth movement** - Spring physics se smooth follow karta hai
2. **Magnetic attraction** - Interactive elements ke paas khichta hai
3. **Repulsion** - Bahut paas jaane par thoda door hatata hai (overlap nahi hota)
4. **Visual feedback** - Hover par bada ho jata hai, glow effect increase hota hai

### Technologies Used (Summary)
1. **React** - Component architecture
2. **TypeScript** - Type safety
3. **Framer Motion** - Advanced animations aur physics
4. **CSS-in-JS** - Dynamic styling
5. **DOM APIs** - Element querying aur positioning
6. **Math/Physics** - Distance calculation, force vectors

### Best Practices Followed
1. **Performance** - Touch devices par auto-disable
2. **Accessibility** - System cursor bhi visible (user ne choose kiya)
3. **Selective Application** - Sirf important elements par magnetic
4. **Smooth Physics** - Natural feeling movement
5. **Visual Hierarchy** - Primary CTAs ko highlight karta hai
